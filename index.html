MellowGlyph gives more "Satisfied" than "Awesome".
You can surely feel it, if possible to look at the details of icon shapes.

Feature

Solid Design

Design of all icons has minimum rounded form to all corner. In addition all icons will fit the square.

Over 550 Icons

That's the pure number of icon artwork. It does not include icon alias.

Developer Friendly

Include many kinds development language icons for developers communication.

Completely Free

MellowGlyph is fully open source. Of course commercial use is also possible.

Vector Graphics

This means that it is smoothly displayed in any size.

Easy To Use

Icon color, size, position and other settings are possible to control on the CSS.

Compatible With All Devices

Possible to high-quality display on mobile phone, laptop or the desktop.

Useful To All-Round

Website, Blog, Apps, Gaming and etc... MellowGlyph will contribute to any use application.

Strict Rules

All of the icon designs has made with strict rules. That made a sense of unity in single collection.
